
React Word Guess Grid

A small React word-guessing game built to practice frontend state management, controlled inputs, conditional rendering, and UI feedback.

The player has five attempts to guess a five-letter word. After each guess, the grid gives feedback for each letter.

Green means the letter is correct and in the correct position.
Yellow means the letter is in the word but in the wrong position.
Red means the letter is not in the word.

Features

5x5 guess grid
Controlled input form
Win and loss states
Letter feedback by position
New game reset button
Random secret word from a small word list

Tech Stack

React
JavaScript
CSS
Vite

What I Practiced

Managing component state with useState
Rendering dynamic rows and cells from arrays
Handling form submission in React
Keeping game logic readable with small helper functions
Using conditional CSS classes for UI feedback
Building and deploying a small frontend project

Run Locally

Clone the repository:
git clone https://github.com/alyssavixp/react-word-guess-grid.git

Go into the project:
cd react-word-guess-grid

Install dependencies:
npm install

Start the development server:
npm run dev

Then open the local URL shown in the terminal.
